# The Knowledge Revision  Engine: MSc Behavioural Science Mastery Guides

This repository is the result of a high-performance synthesis protocol I designed for the MSc Behavioural Science curriculum. It is not a shortcut; it is a **cognitive extension**. All summaries were written with the purpose of quick navigation through a sea of information. Mostly verbatim language is used such to not confuse the reader. The DOIs to the corresponding papers are (or are aimed to be) easily accessible.
